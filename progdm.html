Programavimo kalbos

HTML

Mokslus pradėjau nuo HTML mokymosi. Daug informacijos, o supratimas, apie tai kas kaip ir kur, atėjo po kokio pusmečio. Bet toks, mano nuomone, ir turėtų būti pradedančiųjų WEB programuotoju kelias.

CSS

CSS yra neatskiriama dalis nuo HTML. "Malonumas" atsiranda kada pamatai kaip keičiasi tavo rašomos kodo eilutės į tau matomą web puslapio vizualizaciją. Rašant ir kartojant atsiranda suvokimas.

JavaScript

Be HTML ir CSS , "JavaScript" yra viena iš pagrindinių žiniatinklio programavimų kalbų. "JavaScript" suteikia tinklalapiams gyvumo ir yra svarbi interneto programų dalis.

MySQL

Pradedantieji programuotojai dažniausiai pasirenka būtent MySQL, (duomenų bazių valdymo sistemą), yra būtina kartu su žiniom apie Linux, Apache, MySQL ir PHP.
