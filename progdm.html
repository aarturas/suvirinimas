Programavimo kalbos

HTML

Mokslus BIT'e pradėjau nuo HTML. Daug informacijos, o gilesnis supratimas atėjo po kokio pusmečio. Bet toks, mano nuomone, ir turėtų būti pradedančiųjų WEB programuotojų kelias.

CSS

CSS - neatskiriama nuo HTML. Malonumas atsiranda, kai pamatai, kaip keičiasi tavo rašomos kodo eilutės į tau matomą web puslapio vizualizaciją. Rašant ir kartojant atsiranda suvokimas.

JavaScript

Be HTML ir CSS, JavaScript yra viena iš pagrindinių žiniatinklio programavimų kalbų. JavaScript suteikia tinklalapiams gyvumo ir yra svarbi interneto programų dalis.

MySQL

Pradedantieji programuotojai dažniausiai pasirenka būtent MySQL kuri būtina kartu su Apache ir PHP žiniomis.
